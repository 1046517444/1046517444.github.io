#百度指数
百度指数由搜索指数,媒体指数,资讯指数三部分构成。其对应的接口如下

‘搜索指数’:’http://index.baidu.com/api/SearchApi/index?'
‘媒体指数’: ‘http://index.baidu.com/api/NewsApi/getNewsIndex?',
‘资讯指数’: ‘http://index.baidu.com/api/FeedSearchApi/getFeedIndex?'

后面跟着的参数字典如下

params={
keywords: list, #要搜索的关键字列表，如[‘人工智能’, ‘武林外传’]
start_date: str, #起始时间， 如’2018-01-01’
end_date: str, #结束时间， 如’2019-01-01’
area=0 #0代表全国
}
